## Usage
- Add to your project module [main.bas](main.bas)
- Add to your project class module [дуэлянт.cls](дуэлянт.cls)
- Add to your project class module [револьверСоднимПатроном.cls](револьверСоднимПатроном.cls)
- Run КакЭтоБыло()
